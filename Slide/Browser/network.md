# 網路 Network

--

### 網路協定 (Internet protocol suite)
- 網際網路的基礎通訊架構
  - 由一大堆通訊架構所組成
- 通常被稱為TCP/IP Protocol Suite
  - 因為最早通過的起源架構是TCP和IP
- 分層結構
  - 怎麼分層有不同的分法
  - TCP/IP 4層
    - 應用層 application layer
    - 傳輸層 transport layer
    - 網路互連層 internet layer
    - 網路介面層 link layer
  - OSI 7層
- TCP屬於傳輸層
- IP屬於網路互連層
